Firechrone
=========
🦊

英語/English
---------
Bring compatible Chrome extensions to Firefox with automatic analysis and conversion.
Firechrone is an open-source Firefox extension that helps users install compatible Chrome extensions by analyzing, converting, and adapting them for Firefox whenever possible.

日本語/Japanese
---------
Firechroneとは、Firefoxの拡張機能です。
ChromeWebストアに掲載されているChrome拡張機能をFirefoxで動作させることを目的として開発しています。
ぜひ応援ください。
